Домашнее задание: Работа с CSV в Python

Задача 1: Создай CSV-файл с учениками

Создай список словарей с информацией об учениках:

students = [
    {"name": "Alice", "age": 15, "grade": "A"},
    {"name": "Bob", "age": 14, "grade": "B"},
    {"name": "Charlie", "age": 15, "grade": "C"}
]

Сохрани эти данные в файл students.csv с заголовками и строками.

Задача 2: Прочитай CSV и выведи всех учеников

Создай файл students.csv с содержимым:

name,age,grade
Alice,15,A
Bob,14,B
Charlie,15,C

Прочитай файл и выведи каждого ученика в таком формате:

Имя: Alice, Возраст: 15, Оценка: A

Задача 3: Посчитай средний возраст

Используя файл students.csv, напиши программу, которая считает и выводит средний возраст всех учеников.

Задача 4: Фильтрация данных

Файл students.csv содержит:

name,age,grade
Alice,15,A
Bob,14,B
Charlie,15,C
Diana,14,A
Edward,15,B

Выведи только тех учеников, у кого оценка "A".
