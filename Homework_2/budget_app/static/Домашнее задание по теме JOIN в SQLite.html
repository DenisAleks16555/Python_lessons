Домашнее задание: работа с JOIN в SQLite

Задание 1. Составьте запрос, который выведет названия и цены всех книг, которые когда-либо были заказаны.
Задание 2. Составьте аналогичный запрос, но только для книг, которые ни разу не заказывались.
Задание 3. Найдите суммарную стоимость всех экземпляров книг за авторством А. С. Пушкина, которые находятся на складе.
Задание 4. Определите самую дорогую книгу среди всех проданных.
Задание 5. Посчитайте общее количество книг на складе для каждого автора и выведите результат по каждому из них.
Форма отчёта: выполненные SQL-запросы необходимо прислать в виде отдельных текстовых файлов (*.txt) или одним архивом.
