Домашнее задание: знакомство с APScheduler

Выполните задания, чтобы научиться планировать задачи с помощью библиотеки APScheduler. Используйте разные триггеры (date, interval, cron). Подсказки помогут вам разобраться, а пример вывода подскажет, что должно получиться.

Задание 1. Однократная задача (trigger = date)

Напишите программу, которая через 10 секунд после запуска выводит сообщение "Время вышло!".

Подсказка: используйте datetime.now() и timedelta(seconds=10), а также планировщик BackgroundScheduler.

Пример вывода:

[2025-08-31 16:40:10] Время вышло!

Задание 2. Повторяющаяся задача (trigger = interval)

Создайте задачу, которая каждые 5 секунд выводит текущее время. Пусть программа поработает 20 секунд, чтобы увидеть несколько запусков.

Подсказка: используйте time.sleep(20), чтобы программа не завершилась раньше.

Пример вывода:

[2025-08-31 16:41:00] Сейчас время: 16:41:00
[2025-08-31 16:41:05] Сейчас время: 16:41:05
[2025-08-31 16:41:10] Сейчас время: 16:41:10
[2025-08-31 16:41:15] Сейчас время: 16:41:15

Задание 3. Задача с параметрами

Напишите функцию hello(name), которая выводит приветствие с именем. Запланируйте её запуск раз в 7 секунд, передавая своё имя как параметр.

Подсказка: параметры передаются через аргумент args=['Имя'] в add_job().

Пример вывода:

[2025-08-31 16:42:00] Привет, Аня!
[2025-08-31 16:42:07] Привет, Аня!
[2025-08-31 16:42:14] Привет, Аня!

Задание 4. Использование cron-триггера

Создайте задачу, которая выполняется каждую минуту, но только в течение следующих 3 минут после запуска программы. Пусть выводится сообщение "Напоминание!" с текущим временем.

Подсказка: используйте minute='*/1' в cron-триггере, а остановку приложения через time.sleep().

Пример вывода:

[2025-08-31 16:43:00] Напоминание!
[2025-08-31 16:44:00] Напоминание!
[2025-08-31 16:45:00] Напоминание!

Задание 5. Управление задачами

Создайте две задачи:

  • первая запускается каждые 5 секунд,
  • вторая — каждые 8 секунд.
Через 15 секунд поставьте первую на паузу, через 30 секунд удалите обе задачи.

Подсказка: используйте методы pause_job(), resume_job() и remove_job().

Пример вывода:

[2025-08-31 16:46:00] Задача 1 выполняется!
[2025-08-31 16:46:05] Задача 1 выполняется!
[2025-08-31 16:46:08] Задача 2 выполняется!
... (задача 1 на паузе) ...
[2025-08-31 16:46:16] Задача 2 выполняется!
... (обе задачи удалены) ...
